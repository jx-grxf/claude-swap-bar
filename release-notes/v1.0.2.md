## Highlights

- Fixed a launch crash in packaged builds caused by the SwiftPM resource bundle being installed at the wrong path.
- Added an automated launch check so future release bundles must start successfully before they can pass CI.

## Developer experience

- Added one local command for building, launching, debugging, and verifying the menu bar app.
- Release descriptions now come from reviewed, versioned Markdown files in the repository instead of sparse generated notes.

## Installation

Download the ZIP, move `ClaudeSwapBar.app` to `/Applications`, and open it. Claude Swap Bar is a menu bar app, so it does not open a Dock icon or a regular main window.
