## Highlights

- Added secure automatic updates powered by Sparkle.
- Added Stable and Beta update channels in Settings.
- Added a one-click update check directly in the menu bar popover.

## Security

- Updates are protected by a dedicated Ed25519 signature in addition to Apple Developer ID signing and notarization.
- Sparkle's updater, installer, downloader, and XPC services are signed with the same Developer ID as Claude Swap Bar.

## Installation

Download the ZIP, move `ClaudeSwapBar.app` to `/Applications`, and open it. Future releases can then be installed from Settings → Updates or the update button in the menu bar popover.
